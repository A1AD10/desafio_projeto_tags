Introdução ao HTML na Prática

(Voltar)

Este curso é composto por atividades práticas que introduzem as principais tags e estruturas do HTML. Ele é projetado para iniciantes e foca nos fundamentos essenciais do HTML.

  1. Estrutura Básica de um Documento HTML

    Aprenda a criar a estrutura fundamental de uma página HTML, incluindo as tags <html>, <head>, <title>, <body>, entre outras.

    Compreenda a importância da hierarquia e organização do código para garantir acessibilidade e boas práticas.

  2. Elementos de Texto e Formatação

    Utilize tags como <h1> a <h6> para títulos, <p> para parágrafos, e outras para formatação de texto, como <strong>, <em>, <u>, entre outras.

    Explore a semântica do HTML5 para melhorar a estrutura e acessibilidade do conteúdo.

  3. Listas e Links

    Crie listas ordenadas <ol> e não ordenadas <ul>, utilizando a tag <li> para itens.

    Implemente links de navegação com a tag <a>, aprendendo a definir destinos internos e externos.

  4. Imagens e Multimídia

    Aprenda a inserir imagens utilizando a tag <img>, compreendendo atributos como src e alt.

    Explore a inserção de conteúdos multimídia, como vídeos e áudios, para enriquecer a experiência do usuário.

Trabalhando com Formulários em HTML

(Voltar)

Este curso é uma excelente oportunidade para quem deseja aprofundar seus conhecimentos em desenvolvimento web, focando na criação e manipulação de formulários. Ele aborda os seguintes tópicos essenciais:

  1. Estrutura de um formulário HTML

    Compreenda a construção básica de um formulário utilizando a tag <form>.

  2. Campos de entrada

    Aprenda a utilizar diversos tipos de campos, como <input>, <textarea>, <select>, entre outros, para coletar informações dos usuários.

  3. Validação de dados

    Explore técnicas de validação de dados no lado do cliente, utilizando atributos HTML5 como required, pattern, min, max, entre outros.

  4. Envio de dados

    Entenda os métodos de envio de dados, como GET e POST, e como processá-los adequadamente.

  5. Estilização e acessibilidade

    Aprenda a melhorar a aparência dos formulários com CSS e a garantir que sejam acessíveis a todos os usuários.

Estruturando seu HTML + Formatações

(Voltar)

O curso faz parte da formação HTML Developer, voltada para iniciantes que desejam aprender HTML de forma prática e estruturada. É ideal para quem busca uma introdução sólida ao desenvolvimento web, com foco na criação de páginas bem estruturadas e visualmente agradáveis.

  1. Estruturação de Páginas HTML

    Aprenda a criar a estrutura básica de uma página HTML, utilizando as tags principais como <html>, <head>, <body>, entre outras.

  2. Formatação de Texto

    Utilize tags como <h1> a <h6> para títulos, <p> para parágrafos, e outras para formatação de texto, como <strong>, <em>, <u>, entre outras.

  3. Listas e Links

    Crie listas ordenadas <ol> e não ordenadas <ul>, utilizando a tag <li> para itens. Implemente links de navegação com a tag <a>, aprendendo a definir destinos internos e externos.

  4. Imagens e Multimídia

    Aprenda a inserir imagens utilizando a tag <img>, compreendendo atributos como src e alt. Explore a inserção de conteúdos multimídia, como vídeos e áudios, para enriquecer a experiência do usuário.

  5. Tabelas e Formulários

    Crie tabelas utilizando as tags <table>, <tr>, <td>, entre outras. Desenvolva formulários com campos de entrada, botões e áreas de texto para coletar informações dos usuários.

  6. HTML Semântico

    Utilize elementos semânticos como <header>, <footer>, <article>, entre outros, para melhorar a acessibilidade e a estrutura do conteúdo.

Trabalhando com Mídias utilizando HTML

(Voltar)

Este curso é uma excelente oportunidade para quem deseja aprimorar suas habilidades em desenvolvimento web, focando na integração de mídias em páginas HTML.

  1. Inserção de Imagens

    Utilização da tag <img> para incluir imagens em páginas web.

  2. Integração de Vídeos

    Uso das tags <video> e <source> para incorporar vídeos.

  3. Adição de Áudios

    Implementação da tag <audio> para incluir arquivos de áudio.

  4. Manipulação de Mídias

    Controle de atributos como controls, autoplay, loop, entre outros, para personalizar a reprodução de mídias.

  5. Acessibilidade

    Garantia de que os elementos multimídia sejam acessíveis a todos os usuários, incluindo aqueles com deficiências.

Criando Tabelas com HTML

(Voltar)

Este curso é uma excelente oportunidade para quem deseja aprimorar suas habilidades em desenvolvimento web, focando na criação e estruturação de tabelas em páginas HTML.

  1. Estrutura básica de uma tabela

    Utilização das tags <table>, <tr>, <th> e <td> para criar e organizar tabelas.

  2. Cabeçalhos de tabelas

    Uso da tag <th> para definir células de cabeçalho, proporcionando uma melhor compreensão dos dados.

  3. Linhas e colunas

    Criação de múltiplas linhas e colunas para organizar os dados de forma eficiente.

  4. Atributos de tabelas

    Aplicação de atributos como border, cellspacing, cellpadding, entre outros, para personalizar a aparência das tabelas.

Entendendo HTML Semântico

(Voltar)

Focado na utilização de tags semânticas para criar páginas mais acessíveis, otimizadas para SEO e de fácil manutenção.

  1. Introdução ao HTML Semântico

    Compreensão do que é HTML semântico e seus benefícios.

  2. Elementos semânticos de texto

    Uso de tags como <h1> a <h6>, <p>, <strong>, <em>, entre outras, para formatar textos de forma significativa.

  3. Elementos semânticos de bloco

    Aplicação de tags como <section>, <article>, <aside>, <nav>, entre outras, para estruturar o conteúdo em blocos lógicos.

  4. Outros elementos semânticos

    Utilização de tags como <figure>, <figcaption>, <time>, entre outras, para enriquecer o conteúdo com informações contextuais.

  5. Boas práticas de semântica

    Orientações sobre como utilizar as tags semânticas de forma adequada para melhorar a acessibilidade e o SEO.
